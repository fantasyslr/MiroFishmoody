    1. Create `backend/app/utils/image_helpers.py` with two public functions:

    **`resolve_image_path(image_url: str) -> Optional[str]`**

    - Keep the same security checks: `secure_filename`, `realpath` containment within `IMAGES_DIR`

    **`image_to_base64_part(file_path: str, max_dimension: int = 1024) -> Optional[Dict[str, Any]]`**
    - Read image with `PIL.Image.open(file_path)`
    - If `max(width, height) > max_dimension`: resize with `thumbnail((max_dimension, max_dimension), Image.LANCZOS)`, save to BytesIO as JPEG (quality=85) or PNG depending on original format, base64 encode the BytesIO buffer
    - If `max(width, height) <= max_dimension`: read raw bytes with `open(file_path, "rb")`, base64 encode as-is (same behavior as current `_image_to_base64_part`)
    - Determine MIME from file extension: jpg/jpeg -> "image/jpeg", png -> "image/png", etc.
    - Return `{"type": "image_url", "image_url": {"url": f"data:{mime};base64,{data}"}}`
    - Wrap in try/except, return None on any error, log warning
