**campaign.py API changes:**

4. In `_parse_campaigns()`, read `parent_set_id` from `data.get("parent_set_id")`. If present, store it into each Campaign's `extra["parent_set_id"]` field. This links the new evaluation set to its parent.

5. In `_save_result()`, after saving the JSON, also write `parent_set_id` into the result dict if present in the campaign_set metadata.

6. Modify the `evaluate()` endpoint: read `parent_set_id` from request data. If present, include it in task metadata and in the saved result. Compute version number by walking parent chain (load parent result, check its parent_set_id, count depth + 1). Store `version` and `parent_set_id` in the result dict that gets passed to `_save_result`.

7. Add new endpoint `GET /api/campaign/version-history/<set_id>`:
   - Walk the parent chain backwards from set_id to find the root
   - Walk forward from root to collect all versions
   - Return `{"versions": [{"set_id": "...", "version": N, "created_at": "...", "campaign_names": [...], "overall_scores": {...}}]}`
   - To walk forward: scan all result JSON files in `_RESULTS_DIR`, build a `parent_set_id -> child_set_id` index, then traverse from root.

8. Add new endpoint `GET /api/campaign/compare?v1=<set_id>&v2=<set_id>`:
   - Load both results via `_load_result()`
   - Extract scoreboard campaigns from each
   - For each matching campaign_name across both versions, compute dimension score deltas (v2 - v1)
   - Return:
     ```json
     {
       "v1": {"set_id": "...", "version": N, "scoreboard": {...}},
       "v2": {"set_id": "...", "version": N, "scoreboard": {...}},
       "deltas": {
         "campaign_name": {
           "overall_delta": 0.05,
           "dimension_deltas": {"thumb_stop": 0.1, "clarity": -0.05, ...}
         }
       }
     }
     ```
   - If either result not found, return 404.

Both new endpoints require `@login_required`.

const ITERATE_STATE_KEY = 'mirofishmoody.iterate_state'

  localStorage.setItem(ITERATE_STATE_KEY, JSON.stringify(state))
}
export function getIterateState() {
  const raw = localStorage.getItem(ITERATE_STATE_KEY)
  if (!raw) return null
  try { return JSON.parse(raw) } catch { return null }
}
export function clearIterateState() {
  localStorage.removeItem(ITERATE_STATE_KEY)
}
