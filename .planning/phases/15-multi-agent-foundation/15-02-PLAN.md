```python
"""
AgentScore — 统一 agent 输出 schema

所有新 agent 类型（MultiJudge、DevilsAdvocate、ConsensusAgent 等）
输出此 dataclass，CampaignScorer 自动纳入聚合，无需手工 wiring。

字段：
  agent_type  — agent 类型标识字符串，如 "image_analyzer"、"devil_advocate"
  campaign_id — 对应 campaign ID
  score       — 归一化到 0-1 的贡献分，由 agent 负责归一化
  weight      — 聚合权重，默认 1.0，CampaignScorer 可按 agent_type 覆盖
  metadata    — 扩展字段，存 agent 特有信息（suspect、dissent、visual_score 等）
"""

from __future__ import annotations

from dataclasses import dataclass, field
from typing import Any, Dict


@dataclass
class AgentScore:
    """单个 agent 对单个 campaign 的评分贡献"""
    agent_type: str
    campaign_id: str
    score: float              # 归一化到 0-1
    weight: float = 1.0       # 聚合时的权重
    metadata: Dict[str, Any] = field(default_factory=dict)
```

    - 当 agent_scores 为 None 或空列表时，行为与现在完全一致（向后兼容，不改变任何现有逻辑）
    - 当 agent_scores 非空时，在 `prob_aggregator.aggregate()` 返回 overall scores 之后，将每个 AgentScore 的贡献混入：
      - 按 campaign_id 分组 agent scores
      - 对每个 campaign，计算 weighted_agent_contribution = sum(s.score * s.weight for s in scores) / sum(s.weight for s in scores)（加权均值）
      - 混入比例：AGENT_SCORE_WEIGHT（模块常量，默认 0.1，可通过环境变量 AGENT_SCORE_WEIGHT 配置）
      - 最终 score = existing_score * (1 - AGENT_SCORE_WEIGHT) + weighted_agent_contribution * AGENT_SCORE_WEIGHT
      - 仅对有 agent_scores 的 campaign 做调整，无 agent_scores 的 campaign 保持原分不变
    - 在文件顶部添加 `from ..models.agent_score import AgentScore` import
    - 添加模块级常量：`AGENT_SCORE_WEIGHT = float(os.environ.get('AGENT_SCORE_WEIGHT', '0.1'))`
  </behavior>

4. 在 `score()` 方法体内，找到 `scores = self.prob_aggregator.aggregate(...)` 之后（返回 dict），添加 agent score 混入逻辑：
   ```python
   # --- Agent score contribution (optional) ---
   if agent_scores:
       from collections import defaultdict as _dd
       agent_by_campaign: Dict[str, List[AgentScore]] = _dd(list)
       for a in agent_scores:
           agent_by_campaign[a.campaign_id].append(a)
       for cid, a_list in agent_by_campaign.items():
           if cid not in scores:
               continue
           total_weight = sum(a.weight for a in a_list)
           if total_weight <= 0:
               continue
           weighted_contrib = sum(a.score * a.weight for a in a_list) / total_weight
           scores[cid] = (
               scores[cid] * (1.0 - AGENT_SCORE_WEIGHT)
               + weighted_contrib * AGENT_SCORE_WEIGHT
           )
   ```

5. 添加 `Optional` 到 typing import（如果尚未有）。
